# WorksOnMyMachine

Add sound effects to Android Studio builds.

## Features

🔊 Play sound when build starts  
🔊 Play sound when build succeeds  
🔊 Play sound when build fails  
🔊 Custom sound support

## Default Sounds

If no custom sound is selected, built-in sounds will play.

## Settings

File → Settings → WorksOnMyMachine

## Installation

Install from JetBrains Marketplace or manually via plugin zip.
